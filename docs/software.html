Software

My GitHub page has the most up-to-date list of software projects and repositories. Below are selected research software projects developed by my students, collaborators, and me. Some are current research tools; others are historical packages associated with published work.

Selected projects

  • sdp: deep nonparametric estimation of discrete conditional distributions via smoothed dyadic partitioning. By Wesley Tansey.
  • gfl: a fast and flexible algorithm for solving the graph-fused lasso on an arbitrary graph with arbitrary smooth convex loss. By Wesley Tansey.
  • smoothfdr: methods exploiting spatial structure in multiple-testing problems on spatial lattices or arbitrary graphs. By Wesley Tansey.
  • FDRreg: tools for false-discovery-rate problems, including false-discovery-rate regression.
  • helloPG: an R package skeleton for incorporating the Pólya–Gamma distribution into your own code or package.
  • BayesLogit: Bayesian modeling of discrete outcomes via logistic and negative-binomial regression using Pólya–Gamma data augmentation. Developed by Jesse Windle.
  • BayesBridge: Bayesian bridge regression. Developed by Jesse Windle.
